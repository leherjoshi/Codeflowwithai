Instructions

  1. Make sure your Next.js dev server is running: npm run dev
  2. Open http://localhost:3000
  3. Look for the sun/moon icon in the top-right corner
  4. Click it to toggle between light and dark modes
  5. The entire page should change instantly

✅ What Should Happen

  • Dark Mode: Black background, white text, sun icon visible
  • Light Mode: White/gray background, dark text, moon icon visible
  • Persistence: Theme choice saved and restored on page reload
  • Navigation: Theme persists when navigating between pages

Troubleshooting

  • If button doesn't work, try hard refresh: Cmd+Shift+R or Ctrl+Shift+R
  • Check browser console (F12) for errors
  • Try clearing localStorage using the button above
  • Make sure you're on the latest build: npm run build
